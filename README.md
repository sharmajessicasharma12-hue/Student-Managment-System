# 🎓 Student Management System

## 📖 Description

Student Management System is a Python-based console application that helps manage student records efficiently. Users can add, view, search, update, and delete student information. The project uses JSON file handling to permanently store student data.

---

## ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ✏️ Update Student Details
- ❌ Delete Student
- 💾 Save Data in JSON File
- 📂 Load Data Automatically

---

## 🛠️ Technologies Used

- Python
- Object-Oriented Programming (OOP)
- JSON File Handling
- VS Code
